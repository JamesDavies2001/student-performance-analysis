# Student Performance Analysis

## Overview
This project analyzes student performance data to identify factors associated with academic outcomes. The analysis explores academic, demographic, and lifestyle variables using data cleaning, exploratory visualization, statistical testing, and regression modeling.

## Business / Research Question
Which factors appear to be most strongly associated with student academic performance, and how can these insights help educators or administrators better understand student success?

## Tools Used
- R
- RStudio
- ggplot2
- dplyr
- Statistical testing
- Regression modeling

## Methods
- Data cleaning and preparation
- Exploratory data analysis
- Data visualization
- Correlation analysis
- Statistical testing
- Regression modeling

## Key Findings
- Prior academic performance is the strongest predictor of final grade
- Study time is not a statistically significant predictor of final grade

## Deliverables
- Final report
- Code file / notebook
- Visualizations
- Dataset Documentation

## Why This Matters
-Understanding patterns in student performance can help educators, administrators, and analysts identify factors related to academic success and make more informed decisions about student support.
